MaiaScript is a programming language aimed at developing adaptable and intelligent applications. An adaptive application modifies its code to suit the environment conditions. An intelligent application learns to process the data it receives and adjust to possible changes in its characteristics over time. The MaiaScript language implements the adaptation paradigm, combining it with a variation of the object-oriented paradigm, simplified to exclude type definitions and access restrictions.
